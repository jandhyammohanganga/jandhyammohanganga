## 🚀 Featured Projects

### 📡 APB-Based UART Master Core — RTL & UVM Verification
> *Full-featured UVM environment for UART over APB protocol*

[![Repo](https://img.shields.io/badge/GitHub-Repository-181717?style=for-the-badge&logo=github)](https://github.com/jandhyammohanganga/APB-based-UART-Master-Core-RTL-Verification)

**What I built:**
- Developed a **layered, reusable UVM verification environment** from scratch (Agent, Driver, Monitor, Scoreboard, Coverage Collector)
- Implemented a **Register Abstraction Layer (RAL)** model for register-level access and backdoor reads
- Wrote **constrained random test sequences** targeting corner cases in APB transfers and UART framing
- Authored **SystemVerilog Assertions (SVA)** for protocol compliance and bus timing checks
- Achieved **functional coverage closure** — validating all key functional scenarios

**Tech stack:** `SystemVerilog` · `UVM` · `SVA` · `APB Protocol` · `UART` · `QuestaSim`
